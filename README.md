# Chelsea Medical Marketing

Specialist digital marketing and SEO services for healthcare and medical aesthetics practices.

## About

Chelsea Medical Marketing is a boutique digital marketing agency based in London, specializing exclusively in the healthcare and medical aesthetics sector. We help clinics and practices enhance their online presence, attract more patients, and establish authority through strategic SEO, content marketing, and digital PR.

## Technology

This website is built with [Hugo](https://gohugo.io/) using the [Hugo Fresh](https://github.com/StefMa/hugo-fresh) theme.

## Development

### Prerequisites

- [Hugo Extended](https://gohugo.io/installation/) (v0.146.0 or later)
- [Node.js](https://nodejs.org/) (for PostCSS)

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/chelseahairclinic/chelseahairclinic.co.uk.git
cd chelseahairclinic.co.uk
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
hugo server
```

The site will be available at `http://localhost:1313`

### Build for Production

```bash
npm run build:production
```

## Docker Development

Run the site using Docker:

```bash
docker-compose up
```
